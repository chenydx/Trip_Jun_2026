## Main Decision Rules

1. **6/14 is not a hiking day.** It is a transfer day plus short walk/photo stops.
2. **Taggart Lake is removed from 6/14.** Reconsider only if the group arrives much earlier.
3. **6/15 is the main Grand Teton hiking day.** Jenny Lake and Cascade Canyon stay here.
4. **6/17 and 6/19 are the heavier Yellowstone days.** Do not add long hikes.
5. **Gardiner should be kept if possible.** It works much better than Livingston for Mammoth + Lamar Valley.
