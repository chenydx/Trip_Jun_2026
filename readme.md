## Main Decision Rules

1. **6/14 is not a hiking day.** It is a transfer day plus short walk/photo stops.
2. **Taggart Lake is removed from 6/14.** It can only be reconsidered if the group arrives much earlier and trail/road conditions are favorable.
3. **6/15 is the main Grand Teton hiking day.** Jenny Lake and Cascade Canyon should stay here.
4. **6/19 should prioritize more Yellowstone time, not rushing to Idaho Falls.** Since 6/20 is available for the return drive, skip Idaho Falls and stay closer to Grand Teton / Jackson / Victor / Driggs.
5. **Gardiner should be kept for 6/18 if possible.** It works much better than Livingston for Mammoth + Lamar Valley.
