## 1. Jackson vs Victor / Driggs for 6/14–6/15

### English

**Question:** Should the group stay in Jackson or Victor / Driggs for the Grand Teton portion?

**Jackson** is more convenient but usually much more expensive. It reduces driving after Jenny Lake / Cascade Canyon and gives better sunrise/sunset flexibility.

**Victor / Driggs** is cheaper and still workable, but it requires Teton Pass driving and makes the end of 6/15 more tiring.

Current recommendation:

- Budget priority: **Victor / Driggs**
- Comfort priority: **Jackson**

### 中文

**问题：** 大提顿这两晚住 Jackson，还是住 Victor / Driggs？

**Jackson** 更方便，但通常贵很多。它能减少 Jenny Lake / Cascade Canyon 徒步后的回程压力，也更方便日出日落。

**Victor / Driggs** 更便宜，也能执行这个行程，但需要翻 Teton Pass，6/15 徒步后会更累。

当前建议：

- 预算优先：**Victor / Driggs**
- 舒适优先：**Jackson**

## 2. Yellowstone In-Park Lodging vs Island Park / West Yellowstone

### English

**Question:** Should the group stay inside Yellowstone for 6/16 or 6/17, or keep Island Park?

Current base plan:

- 6/16: Island Park
- 6/17: Island Park
- 6/18: Gardiner

Most useful in-park upgrades:

1. **6/17 Canyon Lodge** — best value upgrade if available. It reduces the long return after Canyon / Hayden Valley.
2. **6/16 Old Faithful area** — helpful for Old Faithful, Upper Geyser Basin, and Grand Prismatic.
3. **6/18 Mammoth** — nice, but less necessary because Gardiner already works well.

Current recommendation:

- If only one Yellowstone in-park night is affordable: **6/17 Canyon Lodge**
- If two nights are affordable: **6/16 Old Faithful area + 6/17 Canyon Lodge**
- Keep **Gardiner** for 6/18 unless Mammoth is close in price.

### 中文

**问题：** 6/16 或 6/17 要不要住黄石园内？还是继续住 Island Park？

当前基础方案：

- 6/16：Island Park
- 6/17：Island Park
- 6/18：Gardiner

最值得考虑的园内升级：

1. **6/17 Canyon Lodge** — 最值得升级，可以减少 Canyon / Hayden Valley 后回 Island Park 的长夜路。
2. **6/16 Old Faithful 区域** — 对 Old Faithful、Upper Geyser Basin、大棱镜更方便。
3. **6/18 Mammoth** — 可以，但不是必须，因为 Gardiner 已经比较顺。

当前建议：

- 只升级一晚：优先 **6/17 Canyon Lodge**
- 升级两晚：**6/16 Old Faithful 区域 + 6/17 Canyon Lodge**
- 6/18 继续住 **Gardiner**，除非 Mammoth 和 Gardiner 价格差不多。
