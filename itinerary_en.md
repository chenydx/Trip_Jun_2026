## Day 1 — 6/14: SLC Arrival + Transfer to Grand Teton Area

This day should be treated as a **transfer day with short photo stops**, not a hiking day.

### Timeline

| Time | Plan |
|---|---|
| 11:00 AM | Pick up rental car at SLC |
| 11:30 AM | Chinatown lunch + quick supply run |
| 1:00 PM | Depart SLC |
| 5:30–6:30 PM | Arrive near Grand Teton / Jackson area, depending on traffic |
| Evening | Schwabacher Landing short walk + Mormon Row if time allows |
| Night | Drive over Teton Pass to Victor / Driggs |

### Recommended Stops

1. **Schwabacher Landing short walk** — 20–45 min
2. **Mormon Row** — 20–30 min

### Removed

- ~~Taggart Lake~~ — removed from 6/14 because the late start makes it too tight and it is not suitable as an evening add-on.

## Day 2 — 6/15: Main Grand Teton Day

Route: **Victor / Driggs → Jenny Lake → Cascade Canyon → Oxbow Bend → Victor / Driggs**

Stop order:

1. Jenny Lake boat round trip / waiting — 0.75–1 hr
2. Hidden Falls + Inspiration Point — 1.5 hr
3. Cascade Canyon short out-and-back only — 2 hr
4. Oxbow Bend at evening light — 0.5 hr
5. Optional backup: Schwabacher Landing or Mormon Row if missed on 6/14

Do not turn Cascade Canyon into a long hike. This day already has hiking plus a return drive over Teton Pass.

---

## Day 3 — 6/16: Yellowstone Southwest

Route: **Victor / Driggs → Grand Teton northbound crossing → Yellowstone Southwest → Island Park**

Stop order:

1. Old Faithful + Upper Geyser Basin — 2.5–3 hr
2. Grand Prismatic Overlook / Fairy Falls Trail — 1.5 hr

Do not add extra Grand Teton sightseeing here. This is a transfer plus Yellowstone sightseeing day.

---

## Day 4 — 6/17: Yellowstone Lower Loop

Route: **Island Park → Norris → Canyon / Artist Point → Hayden Valley → Island Park**

Stop order:

1. Norris Geyser Basin — 1.5 hr
2. Grand Canyon of the Yellowstone / Artist Point — 2 hr
3. Hayden Valley wildlife viewing — 1–1.5 hr

Do not stay too late in Hayden Valley because the return to Island Park is long.

---

## Day 5 — 6/18: Yellowstone North Transfer Day

Route: **Island Park → Mammoth → Gardiner → Lamar Valley → Gardiner**

Stop order:

1. Mammoth Hot Springs — 1.5 hr
2. Gardiner check-in / dinner — 1.5 hr
3. Lamar Valley wildlife viewing — 1.5–2 hr

Gardiner is the recommended lodging base for this night. It is much better than Livingston for this itinerary.

---

## Day 6 — 6/19: More Yellowstone Time + South Entrance Exit

Route: **Gardiner → Mammoth / Canyon corridor → Yellowstone Lake → West Thumb → South Entrance → Jackson / Victor / Driggs**

Stop order:

1. Optional backup: Hayden Valley or Mud Volcano if timing works
2. Yellowstone Lake scenic pullouts — 0.5–1 hr
3. West Thumb Geyser Basin — 1–1.25 hr
4. Exit via the South Entrance
5. Continue toward Jackson / Victor / Driggs

### Why This Changed

The previous plan sent the group to Idaho Falls on 6/19. That made 6/20 easier, but it rushed the final Yellowstone day.

Since the group has all day on 6/20 to return to SLC, it is better to spend more time in Yellowstone on 6/19 and stay around Jackson / Victor / Driggs instead.

This is still a long day, but it is a better use of park time. Do not add a long hike.

---

## Day 7 — 6/20: Long Return to SLC

Route: **Jackson / Victor / Driggs → Salt Lake City**

Drive:

- From Jackson / Victor / Driggs to SLC: about **4.5–5.5 hr**
- Add extra buffer for meals, gas, weather, roadwork, and rental car return.

Plan:

1. Drive south toward Salt Lake City.
2. Optional quick photo stop only if the departure schedule allows.
3. Return rental car.
4. Depart from SLC or rest in the city if staying longer.

This plan only works well if the group truly has most of the day on 6/20. If there is an early flight, the old Idaho Falls plan is safer.
